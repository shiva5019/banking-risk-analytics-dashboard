# Banking Risk Analytics Dashboard

## Project Overview
The Banking Risk Analytics Dashboard project focuses on analyzing banking customer data to generate meaningful business insights related to loans, deposits, customer segmentation, and financial performance.

This project combines Python-based Exploratory Data Analysis (EDA) with interactive Power BI dashboards to support business decision-making and risk analysis.

---

# Problem Statement
Banks generate large amounts of customer and financial data every day. Analyzing this data manually is difficult and time-consuming.

The objective of this project is to:
- Analyze customer banking behavior
- Identify loan and deposit trends
- Understand customer segmentation
- Monitor important banking KPIs
- Create interactive dashboards for better decision-making
